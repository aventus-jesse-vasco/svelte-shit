📘 Samenvatting van de theorie

Svelte is een modern webframework waarmee je websites en webapps maakt — een beetje zoals React of Vue, maar met een groot verschil: In plaats van veel code te draaien in de browser, vertaalt (compileert) Svelte jouw code van tevoren naar supersnelle, gewone JavaScript.

Daardoor is je website sneller, zijn de bestanden kleiner en schrijf je minder code.



Je werkt met componenten — kleine bouwblokken met HTML, CSS en JavaScript in één bestand (.svelte). Als je iets aanpast in je data, past de gebruikersinterface zich automatisch aan — dat heet reactiviteit.

📝 Instructie – Praktijkopdracht

Je zet het Svelte project op en voegt de eerste herbruikbare componenten toe. Je gebruikt Tailwind om de componenten te stylen.



1.        Maak een git repo aan via https://classroom.github.com/a/aV5b9L4e

2.        Maak een nieuw Svelte-project

npx sv create

SvelteKit minimal

Typescript

Add: prettier, eslint, tailwindcss

Tailwind: typography and forms

Package manager: npm



3.        Start het project

npm run dev -- --open



4.        Maak een component  Welcome.svelte in src/lib/components

Toon het component op de homepage (routes/page.svelte)
5.        Maak een component InfoCard.svelte met een stukje uitleg over jezelf of je huisdier.

Toon het component op de homepage
Gebruik hierbij styling dmv Tailwind
