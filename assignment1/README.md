# SVLT01 - Assignment 1: Introductie & Componenten

## Setup

```bash
npm install
npm run dev -- --open
```

## Wat is gedaan

- SvelteKit minimal + TypeScript + Tailwind (forms, typography)
- `src/lib/components/Welcome.svelte` - begroeting component met prop `name`
- `src/lib/components/InfoCard.svelte` - kaart met informatie over huisdier/zelf
- Beide componenten getoond op `src/routes/+page.svelte` en gestyled met Tailwind
